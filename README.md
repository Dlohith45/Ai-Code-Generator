# AI Code Generator

## Overview

AI Code Generator is a web-based application that generates programming code templates based on user input. Users can enter a programming requirement, choose a programming language, and instantly receive the corresponding source code.

This project demonstrates frontend-backend integration using HTML, CSS, JavaScript, and Python Flask.

---

## Features

* Generate code using natural language prompts
* Supports multiple programming languages

  * Java
  * Python
  * JavaScript
* Copy generated code
* Download generated code
* Clear input and output
* Clean and responsive user interface
